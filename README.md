# Sales Performance Analytics Dashboard (Interactive Dashboard Created Using MS Excel)
## Project Objective
To design an interactive Excel-based dashboard that transforms raw sales transaction data into meaningful business insights, enabling analysis of regional performance, product contribution, salesperson effectiveness, customer segments, and sales trends for better data-driven decision-making.

## Dataset Used
Link :- https://excelx.com/practice-data/sales-retail/

## Questions (KPIs)
• Which region has the highest sales?

• Which products are selling the most?

• Who are the top salespersons based on sales value?

• How are sales changing month by month?

• Which customer type contributes more to overall sales?

• What is the overall sales performance?

• How can sales data be quickly analyzed using filters?

• How can management track sales performance from one dashboard instead of multiple reports?

## Process 
• Collected and cleaned raw sales data, ensuring proper formatting and structured data for analysis.

• Built pivot tables to analyze sales performance by region, product, salesperson, customer type, and time trends.

• Created KPI cards and visual charts to represent key business metrics in an easy-to-understand format.

• Designed an interactive dashboard using slicers and proper layout alignment to enable dynamic analysis and decision-making.

## Project Insights

• Regional analysis showed clear differences in sales performance, helping identify high-performing and low-performing areas.

• Product analysis highlighted which products contribute most to total sales, indicating key revenue drivers.

• Salesperson performance comparison helped identify top contributors and overall sales distribution.

• Monthly trend analysis showed patterns in sales performance over time, helping understand growth and fluctuation.

• Customer type analysis provided insight into which customer segment contributes more to overall sales.

• KPI cards provided a quick summary of overall business performance, making monitoring easier.

## Final Conclusion

The Sales Performance Dashboard successfully transformed raw sales data into a clear, interactive, and easy-to-understand analytics solution. By combining pivot tables, KPI metrics, and visual charts, the dashboard enables quick performance tracking across regions, products, salespersons, and customer segments.Overall, the project demonstrates how Excel can be used to build a structured, data-driven reporting system that improves visibility and supports faster business decision-making.
